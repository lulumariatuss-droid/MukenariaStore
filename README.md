Website e-commerce **Mukenaria** merupakan aplikasi web yang dibuat menggunakan **HTML, CSS, dan JavaScript murni (Vanilla JavaScript)** tanpa framework tambahan. Website ini dirancang untuk memberikan pengalaman berbelanja mukena secara modern, elegan, responsif, dan interaktif.

---

**Mukenaria** adalah website e-commerce yang menyediakan berbagai koleksi mukena modern dengan desain elegan, bahan berkualitas, serta nyaman digunakan untuk beribadah. Website ini dirancang agar pelanggan dapat mencari, memilih, hingga melakukan pembelian produk secara mudah melalui tampilan yang responsif dan user-friendly.

## 🎯 Value Proposition

Mukenaria menawarkan nilai lebih dibandingkan toko mukena pada umumnya, yaitu:

- Menggunakan bahan premium yang lembut, adem, dan nyaman dipakai.
- Desain modern namun tetap sesuai dengan nilai-nilai islami.
- Tersedia berbagai pilihan warna, ukuran, dan kategori produk.
- Tampilan website elegan dan mudah digunakan.
- Checkout cepat dengan berbagai metode pembayaran.
- Pelayanan pelanggan yang responsif.
- Promo mahasiswa dan gratis ongkir untuk wilayah tertentu.

# 🛒 Proses Checkout & Simulasi Payment Gateway

## Alur Checkout

1. Pilih produk.
2. Tambahkan ke keranjang.
3. Isi data pengiriman.
4. Masukkan kode promo (opsional).
5. Pilih metode pembayaran.
6. Konfirmasi pesanan.
7. Pesanan diproses.

## Simulasi Payment Gateway

Website menggunakan simulasi **Midtrans Sandbox**.

Metode pembayaran yang disediakan:

- QRIS
- Transfer Bank (Virtual Account)
- E-Wallet
- Kartu Kredit/Debit

### Alur Simulasi Midtrans

```
Checkout
      ↓
Website Mukenaria
      ↓
Midtrans Sandbox
      ↓
Pilih Metode Pembayaran
      ↓
Pembayaran Berhasil
      ↓
Status Order = Paid
```
